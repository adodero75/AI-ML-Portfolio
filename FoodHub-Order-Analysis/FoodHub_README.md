# FoodHub Order Analysis

## Business Problem
FoodHub, a food aggregator app, needs to analyze order data to improve customer experience and optimize operations.

## Solution Approach
Performed data cleaning, univariate and multivariate analysis, and answered business questions using Python.

## Techniques Used
Data wrangling, EDA (histograms, boxplots), grouping and aggregation, correlation analysis.

## Key Insights
Delivery speed has minimal impact on ratings; food quality matters more. Cuisine type influences ratings and revenue.

## Recommendations
Promote high-rated cuisines (Indian, Thai, Spanish), optimize delivery logistics, and improve low-rated cuisines.
